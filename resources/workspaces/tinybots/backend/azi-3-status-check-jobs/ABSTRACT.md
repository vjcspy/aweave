Stateless Node.js worker that schedules per-robot, per-rule monitoring windows and runs actions (currently alarms) when gaps in events are detected. Background job service for AZI-3 status checks: creates daily monitoring sessions for configured robots and raises actions (alarms) when expected events are absent within configured rolling windows.

Internal admin dashboard for TinyBots staff to manage robots, TaaS orders, organisations, subscriptions, and voice assistant devices. Supports order lifecycle management, device enrollment, and analytics. The app communicates with TinyBots admin APIs. URLs are configured in src/app/common/utilities/constants/constants.module.ts.

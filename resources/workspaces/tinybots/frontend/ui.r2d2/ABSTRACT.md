Customer-facing React web application for managing Tessa robots. Users can schedule tasks, manage music playlists, configure speech interactions, and control robot settings. The app communicates with multiple TinyBots backend microservices via REST APIs. API URLs are configured in src/common/constants/urls/urls.json.
